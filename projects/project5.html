Budget Manager

This application was desinged to ease money management for the users. It is a single user based application build on Android 2.2 system. The application had features to enter the monthly budget and the expenses for corresponding month are recorded. The monthly budget were shown in three formats viz. (a) a list format,(b) a pie-chart of expenses and (c)a final tally of the entire expense v/s budget.
 The system is now a legacy one, as we didn't get further scope to update the project.

Powerful Code

Light-weight Database

Since the system is to be used by a single user, we have used SQLite database to ease the access and process the data quickly within the system resulting in synchronized performance of the application.

project summary

The mobile application was developed as per the requirement specification. It is currently launced in slideme.org and it has over 3000+ downloads. It has developed in Java, XML and SQLite as a custom MVC based PHP framework, the system is thoroughly tested as per industry standards.

Summary Details

It is one of the most downloaded budget management application in Slideme.org(a free application portal for Android). Considering the fact that it is one of our first build application, it has performed quite well and beyond our expectations.
